xieite::unroll()

 Defined in fragment xieite:unroll

Unrolls a constant-sized loop using std::integer_sequence {} and std::make_integer_sequence {}.
Passes the indices to a given functor as a NTTP pack.


Declaration
template<std::integral auto n>
constexpr auto unroll(auto&& fn, auto&&... args)
XIEITE_ARROW(/* XIEITE_FWD(fn).template operator()<0, 1, 2, ..., n>() */)


Example
import xieite;

int main() {
    xieite::unroll<5>([]<int... i> {
        xieite::dump(i...);
    });
}
Output:
0 1 2 3 4
