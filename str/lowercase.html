xieite::lowercase()

 Defined in fragment xieite:lowercase

Converts every character of a string (or a single character) to lowercase.
Only handles the English alphabet.


Declarations
#0
[[nodiscard]] constexpr char lowercase(char c) noexcept;
#1
[[nodiscard]] constexpr std::string lowercase(std::string_view str) noexcept;


Example
import xieite;

int main() {
    xieite::dump(xieite::lowercase("HeLlO, wOrLd!"));
}
Output:
hello, world!
