xieite::dump()

 Dumps values to stdout, separated by single spaces.
Essentially just std::println() with the format generated automatically.
Based on P2879R0.


Declaration
template<typename... Args>
void dump(Args&&... args) noexcept;


Example
import std;
import xieite;

int main() {
    int x = 418;
    std::string_view y = "hello";
    double z = 3.14159;

    xieite::dump(x, y, z);
}
418 hello 3.14159
