xieite::rotate_swap()

 Swaps the values of multiple references by rotating values to the left.
When rotating multiple spaces, internally uses the least possible number of temporary copies.

Declaration
template<std::size_t n = 1, typename... Args>
constexpr auto rotate_swap(Args&... args);
 Returns a discardable std::tuple of references.


Example
import std;
import xieite;

int main() {
    int a = 1;
    int b = 2;
    int c = 3;
    xieite::dump(a, b, c);

    xieite::rotate_swap(a, b, c);

    xieite::dump(a, b, c);
}
1 2 3
2 3 1
