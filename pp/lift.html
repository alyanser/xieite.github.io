XIEITE_LIFT(), XIEITE_LIFT_LOCAL()

 Defined in header <xieite/lift.hpp>

Wraps a function in a lambda object.


Definitions
#0
#define XIEITE_LIFT(...) /* ... */
Expands to a lambda with no capture-default. Cannot access local variables.

#1
#define XIEITE_LIFT_LOCAL(...) /* ... */
Expands to a lambda with a reference capture-default. Cannot be used in global scope.


Example
#include <xieite/fwd.hpp>
#include <xieite/lift.hpp>
import std;

void run(auto&& fn) {
    std::invoke(XIEITE_FWD(fn), "Hello, world!");
}

int main() {
    run(XIEITE_LIFT(std::println));
}
Output:
Hello, world!
